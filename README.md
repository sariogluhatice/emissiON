# 🌍 EmissiON

**EmissiON** is a web-based carbon footprint management platform developed as a senior capstone project at Işık University.

The system enables **individuals, households, and companies** to monitor, analyze, and reduce their carbon emissions through data-driven insights, AI-assisted recommendations, and sustainability tracking tools.

## Key Features

### 👤 Individual Carbon Tracking

* Record and monitor personal carbon emissions.
* Visualize emission trends through interactive dashboards.
* Receive AI-generated sustainability recommendations.

### 🏠 Household Management

* Create and manage household groups.
* Assign sustainability tasks to members.
* Track collective environmental impact.

### 🏢 Corporate Sustainability & CBAM

* Monitor organizational emissions.
* Generate CBAM (Carbon Border Adjustment Mechanism) reports.
* Analyze emission performance using industry-specific metrics.

### 🤖 AI-Powered Insights

* Emission analysis using Large Language Models.
* Personalized reduction roadmaps.
* Sustainability improvement suggestions.

### 📄 Smart Document Processing

* Automatic extraction of emission data from uploaded documents.
* OCR integration using AWS Textract.
* Automated categorization and validation of records.

## System Architecture

EmissiON follows a modern web architecture consisting of:

* Frontend: Vanilla JavaScript
* Backend: Node.js & Express.js
* Database: PostgreSQL 16
* Cloud Storage: AWS S3
* OCR Service: AWS Textract
* AI Services: Groq & Gemini
* Emission Data Provider: Climatiq API
